**🐦 Flappy Bird (Python + Pygame)**

A simple recreation of the classic Flappy Bird game built using Python and Pygame.
This project demonstrates basic game physics, collision detection, and event handling.

⸻

**🎮 Demo**



⸻

**🚀 Features**
	•	Gravity-based bird movement
	•	Pipe obstacles
	•	Collision detection
	•	Score tracking

⸻

**🛠️ Tech Stack**
	•	Python
	•	Pygame

⸻

**🧠 What I Learned**
- How to design a **game loop** in Python
- Implementing **physics simulation (gravity & velocity)**
- Handling **user input events** in real time
- Building **collision detection logic**
- Structuring a small project for scalability

⸻

**📦 Installation**

git clone https://github.com/JakeLee111/Flappy-bird-Recreation.git
cd Flappy-bird-Recreation
pip install pygame
python main.py

**🎯 How to Play**
	•	Press SPACE to jump
	•	Avoid hitting pipes
	•	Try to beat your high score
